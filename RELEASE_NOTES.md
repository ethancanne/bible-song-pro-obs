# Bible Song Pro - Version 1.0

## Release Summary

Bible Song Pro `Version 1.0` is the initial public release of the plugin.

Built for OBS custom dock workflows, it gives churches and live presentation teams a unified control panel for Bible verses, song lyrics, setlists, styling, and projection.

## Highlights

- Auto-Retrieve Lyrics
- Dual Bible Version support
- Multi-language interface
- Support for 50+ languages
- Support for 250+ Bible versions
- Full Screen and Lower Third modes
- Flexible background and typography controls
- Quick Actions panel
- Theme support
- OBS-centered workflow

## Included Download

The release zip contains:

- `Bible Song Pro panel.html`
- `BSP_display.html`

## Notes

- Designed for OBS custom dock use
- Keep both HTML files in the same folder after extraction
- Test your workflow in OBS before live use
